# 项目介绍

这是一个名为 Delightify Exporter 的项目，旨在为用户提供易于使用的工具，以便导出 Delightify 中的数据。

## 命令

使用以下命令导出数据:
```
/delightify_export dump
```

## 输出路径

数据会被导出到以下路径:
```
<world>/delightify-exporter/export.sqlite
```
